VS Code Shortcuts

Opening and Saving

These shortcuts help you quickly open, create, and save files in VS Code.

They are essential for working efficiently without the mouse.

  • Ctrl + N — New File
  • Ctrl + O — Open File
  • Ctrl + S — Save File
  • Ctrl + K S — Save All Files
  • Ctrl + W — Close File
Editing

Use these shortcuts to edit text faster and manage your code easily.

They help minimize repetitive actions and improve your overall coding speed.

  • Ctrl + X — Cut Line
  • Ctrl + C — Copy Line
  • Ctrl + V — Paste Line
  • Ctrl + Z — Undo
  • Ctrl + Y — Redo
Navigation

Navigation shortcuts let you move through files and tabs easily.

They reduce the need for mouse interaction, keeping your hands on the keyboard for better efficiency.

  • Ctrl + Tab — Switch Tabs
  • Ctrl + P — Quick Open File
  • Ctrl + G — Go to Line
  • Ctrl + Home — Go to Start
Emmet Abbreviations

Emmet expands short text into full HTML structures instantly.

It’s a powerful way to speed up writing repetitive HTML or CSS code snippets.

  • ! — Expands to full HTML boilerplate
  • ul>li*3 — Creates a list with three items
  • div>p — Creates a div with a paragraph inside
  • Tab — Expands abbreviation
Terminal

The integrated terminal allows you to run commands directly in VS Code.

It keeps your development environment unified, saving time switching between external terminals.

  • Ctrl + ` — Toggle Terminal
  • Ctrl + Shift + ` — New Terminal
  • Ctrl + L — Clear Terminal
  • Ctrl + Shift + W — Close Terminal
